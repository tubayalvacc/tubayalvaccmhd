Featured Projects

Wheels N’ Works Web Application

A comprehensive vehicle repair workshop management web application developed during my full-stack development internship. Designed to streamline repair shop operations for customers, shop owners, and super admins through dedicated role-based dashboards.


The application allows customers to book appointments, view booking status, and receive invoices. Shop owners can manage repair orders, customer information, mechanic assignments, repair status updates, and billing. Super admin panel provides shop performance monitoring, system management, and access control.


Key Features:

  • Multi-user system: Customer, Shop Owner (Admin), and Super Admin dashboards
  • Appointment booking, repair job management, invoicing
  • Secure login with session management for different roles
  • RESTful APIs for data handling and inter-page communication
  • Responsive front-end design for web and mobile browsers

My Role: Designed and developed the complete system architecture, front-end pages using HTML, CSS, and JavaScript, back-end logic in PHP, and MySQL database management. Integrated APIs for seamless data flow and implemented session-based access control. Uploaded and versioned code using GitHub for collaborative tracking.


Tech Stack: HTML, CSS, JavaScript, PHP, MySQL, RESTful API, GitHub

Robotic Arm (Object Picker)

An Arduino-controlled robotic arm capable of precise multi-axis movement using 6 servo motors, controlled via a servo motor driver module. The arm can pick and place small objects and perform automated sorting tasks. This was my first major robotics build involving mechanical assembly, servo synchronization, and power management.


Key Features:

  • 6 servo motor-controlled joints for flexible movement
  • Servo driver module for stable multi-servo control
  • Custom-designed movement sequences via Arduino IDE

Tech Stack: Arduino IDE, MG996 Servo Motors, Servo Motor Driver, C++

Line Follower & Obstacle Avoiding Robot with Robotic Arm

A multifunction Arduino-powered robot designed to follow a line using IR sensors, detect and avoid obstacles with ultrasonic sensors, and pick/place objects using a servo-based robotic arm. Includes an LCD screen for real-time status display of sensor readings and robot actions.


Key Features:

  • Line following using IR sensors
  • Obstacle detection using an ultrasonic distance sensor
  • Object-picking robotic arm with 6 servo motors
  • Johnson motors controlled via motor driver modules
  • LCD screen displaying system status and distance readings

Tech Stack: Arduino IDE, Johnson Motors, L298N Motor Driver, IR Sensors, Ultrasonic Sensor, 6 Servo Motors, LCD 16x2 Display, C++
